SID Register & Frequenztabelle

$D400 – $D41CMOS 6581 / 85803 Stimmen + Filter

Der SID (Sound Interface Device) liegt bei $D400–$D41C und spiegelt sich bis $D7FF. Jede der 3 Stimmen hat 7 Register. Frequenz-Formel (PAL): Fn = Fout × 16777216 / 985248

Stimmen-Register (Stimme 1: $D400–$D406, Stimme 2: $D407, Stimme 3: $D40E)

OffsetStimme 1Stimme 2Stimme 3Funktion
+0$D400$D407$D40EFrequenz Low-Byte (Bits 7–0)
+1$D401$D408$D40FFrequenz High-Byte (Bits 15–8)
+2$D402$D409$D410Pulsbreite Low (Bits 7–0) — nur Rechteck-Wellenform
+3$D403$D40A$D411Pulsbreite High (Bits 3–0) — 0=$000 (0%), $800=50%, $FFF=100%
+4$D404$D40B$D412Control-Register (Bits, s.u.)
+5$D405$D40C$D413Attack/Decay: Bits 7–4 = Attack (0–15), Bits 3–0 = Decay (0–15)
+6$D406$D40D$D414Sustain/Release: Bits 7–4 = Sustain-Level (0–15), Bits 3–0 = Release (0–15)

Control-Register Bits (Offset +4)

BitNameFunktion
0GATE1 = Note starten (Attack beginnt). 0 = Note stoppen (Release beginnt)
1SYNCFrequenz von Stimme N mit Stimme N-1 synchronisieren (1→2→3→1)
2RINGRing-Modulation: Dreieck-Welle moduliert mit Nachbar-Stimme
3TESTWaveform-Generator zurücksetzen (Phasen-Reset)
4TRIDreieck-Wellenform aktivieren
5SAWSägezahn-Wellenform aktivieren
6PULRechteck (Puls) -Wellenform aktivieren (Pulsbreite via +2/+3)
7NOIRauschen aktivieren (Pseudo-Random, 23-Bit LFSR)
Wellenform-Kombinationen Mehrere Wellenform-Bits gleichzeitig = UND-Verknüpfung (dunklerer Klang). Klassisch: SAW+TRI ($60) für weichen Lead. Nur TRI ($10) für reine Sinusnäherung. Nur NOI ($80) für Schlagzeug/Rauschen.

AdresseBitsFunktion

          LDA #$64 : STA $D400   ; Freq Lo = $64
          LDA #$11 : STA $D401   ; Freq Hi = $11
          LDA #$00 : STA $D405   ; AD: A=0ms D=6ms
          LDA #$F0 : STA $D406   ; SR: S=15 R=6ms
          LDA #$41 : STA $D404   ; Puls-Wellenform + GATE=1 (start)
          LDA #$0F : STA $D418   ; Lautstärke = 15 (max)
