CIA Register & Tastaturmatrix

CIA1: $DC00–$DCFFCIA2: $DD00–$DDFFMOS 6526

Zwei CIA-Chips (Complex Interface Adapter) steuern Tastatur, Joysticks, serielle Schnittstelle, Timer und Interrupts. CIA1 bei $DC00, CIA2 bei $DD00. Jeder hat 16 Register (gespiegelt bis +FF).

ICR — Interrupt Control Register ($DC0D / $DD0D)

Lesen: gibt Interrupt-Status zurück (Bit 7=IRQ ausgelöst, Bits 4–0=Quellen). Lesen löscht alle gesetzten Bits!
Schreiben: Bit 7=1 → gesetzte Bits in Maske aktivieren. Bit 7=0 → gesetzte Bits deaktivieren.

CRA / CRB — Control Register ($DC0E/$DC0F)

BitNameFunktion
0START1 = Timer starten, 0 = stoppen
1PBONTimer-Ausgang auf Port B sichtbar machen
2OUTMODE0 = Toggle, 1 = Puls auf Port B
3RUNMODE0 = Continuous (Neustart nach Unterlauf), 1 = One-Shot

5INMODECRA: 0=Phi2-Takt, 1=CNT-Pin. CRB: 00=Phi2, 01=CNT, 10=TimerA-UF, 11=TimerA-UF+CNT

7TODINTOD-Takt: 0=60Hz, 1=50Hz
 
; Timer A auf 50Hz konfigurieren (PAL: 985248 / 50 = 19704 = $4CC8)
          LDA #$C8 : STA $DC04  ; Timer A Low = $C8
          LDA #$4C : STA $DC05  ; Timer A High = $4C
          LDA #$7F : STA $DC0D  ; Alle CIA-IRQs deaktivieren
          LDA #$81 : STA $DC0D  ; Timer A IRQ aktivieren (Bit7=set, Bit0=TA)
          LDA #$11 : STA $DC0E  ; LOAD + START (Bits 4+0)

; Joystick Port 2 lesen ($DC00)
ReadJoy2:
          LDA $DC00
          EOR #$FF       ; Invertieren (active LOW → active HIGH)
          AND #$1F       ; Bits 0-4: Up/Down/Left/Right/Fire
          RTS

; Tastatur scannen: Prüfe ob SPACE gedrückt
; SPACE ist in Spalte 4, Zeile 7
CheckSpace:
          LDA #%11101111  ; Alle Spalten ein, außer Spalte 4 (Bit 4=0)
          STA $DC00
          LDA $DC01       ; Zeilen lesen
          AND #%10000000  ; Zeile 7 prüfen
          BEQ .pressed    ; 0 = gedrückt (active LOW)
          CLC : RTS       ; Nicht gedrückt
.pressed: SEC : RTS       ; Gedrückt
