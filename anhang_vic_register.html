VIC-II Register-Referenz

$D000 – $D02E47 Register6569 PAL / 6567 NTSC

Der VIC-II (MOS 6569 PAL, 6567 NTSC) liegt bei $D000–$D02E. Der gesamte I/O-Bereich $D000–$D3FF spiegelt den VIC alle 64 Bytes. Zugriff nur bei aktivem I/O ($01 Bits 0–2 ≠ %100).

$D018 Bits 7–4 (VM13–VM10)Screen-RAM Adresse (in VIC-Bank)
%0000 = 0+$0000 (z.B. $0000 in Bank 0)
%0001 = 1+$0400 ← Standard Screen-RAM
%0010 = 2+$0800
%0100 = 4+$1000
%1000 = 8+$2000
%1100 = 12+$3000
$D018 Bits 3–1 (CB13–CB11)Charset/Bitmap Adresse (in VIC-Bank)
%000 = 0+$0000
%001 = 1+$0800
%010 = 2+$1000 ← Standard Charset (ROM in Bank 0/2)
%011 = 3+$1800
%100 = 4+$2000 ← Bitmap-Startadresse 1
%110 = 6+$3000
$D018 Beispielwerte $10 = Standard (Screen=$0400, Charset=$1000 ROM) | $18 = Screen=$0400, Bitmap=$2000 | $78 = Screen=$1C00, Bitmap=$2000 (typisch FLI)

; Häufige VIC-II Konfigurationen

; Standard-Zeichenmodus (bereits aktiv nach Reset):
          LDA #$1B : STA $D011   ; ECM=0 BMM=0 DEN=1 RSEL=1 YSCROLL=3
          LDA #$C8 : STA $D016   ; MCM=0 CSEL=1 XSCROLL=0
          LDA #$14 : STA $D018   ; Screen=$0400 Charset=$1000

; Hires Bitmap Mode:
          LDA #$3B : STA $D011   ; BMM=1 (Bit 5 gesetzt)
          LDA #$C8 : STA $D016
          LDA #$18 : STA $D018   ; Screen=$0400 Bitmap=$2000

; Raster-IRQ auf Zeile 100 setzen:
          LDA #100 : STA $D012
          LDA $D011 : AND #$7F : STA $D011  ; RST8=0 (Zeile < 256)
          LDA #$01 : STA $D01A   ; Raster-IRQ enable
