VICE-Konfiguration einrichten

Empfohlene VICE-Einstellungen aus Stufe 00 als fertige Konfigurationsdatei: 

# Linux/macOS:
cp ../vice.vicerc ~/.config/vice/vicerc

# Oder direkt beim Start:
x64sc -config ../vice.vicerc meinprogramm.prg

# Windows: Datei nach %APPDATA%\vice\vicerc kopieren

Schnellstart — Codebeispiel kompilieren und ausführen

Voraussetzung: ACME Assembler und VICE Emulator installiert.

# Beispiel aus Stufe 01 kompilieren:
cd stufe_01/
acme --cpu 6510 -f cbm -o beispiel.prg 3_1_lade_und_speicherbefehle_load_store.asm

# In VICE öffnen:
x64sc beispiel.prg

# Alle .asm Dateien einer Stufe auf einmal kompilieren:
for f in *.asm; do
  acme --cpu 6510 -f cbm -o "${f%.asm}.prg" "$f" && echo "OK: $f"
done

Jede .asm-Datei enthält einen Kommentar-Header mit dem genauen ACME-Aufruf. Nicht alle Beispiele sind eigenständige Programme — manche zeigen nur einzelne Techniken als Codeausschnitt und müssen in einen vollständigen Rahmen eingebettet werden.
